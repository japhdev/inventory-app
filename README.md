## 🛠️ Technologies Used

**Frontend**
- React.js — component-based UI for managing products and inventory
- JavaScript — client-side logic and API requests
- JSX — templating syntax for React components
- HTML5 — semantic structure
- CSS3 — styling

**Backend**
- Python — backend development
- FastAPI — high-performance REST API framework
- Uvicorn — ASGI server for running the FastAPI application

**Database**
- PostgreSQL — relational database for storing product data

**Security**
- JWT (JSON Web Tokens) — authentication system
- bcrypt & passlib — secure password hashing
- datetime — token expiration and time handling
---

## ✨ Features

- Full CRUD system to manage inventory (create, edit, and delete products)
- PostgreSQL database integration for persistent storage
- Sales panel to search products, register sales, and automatically update inventory stock
- Real-time synchronization between the application and the database
- Low stock alert system to notify when products are running out
- Toggle banner used as a visual reminder
- Product status system (active / inactive) when stock reaches 0
- Authentication system in progress with Login and Register using JWT for improved security
- Scalable architecture prepared for future improvements and additional features
